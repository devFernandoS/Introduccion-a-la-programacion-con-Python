# Curso Santander | Introducción a la programación con Python

Bienvenido al curso de Introducción a la programación con Python, ofrecido por Santander. Este curso está diseñado para personas que desean aprender los fundamentos de la programación utilizando Python, uno de los lenguajes más populares y versátiles en la actualidad.

## Contenidos del curso

El curso está dividido en módulos que cubren los siguientes temas:

### Módulo 1: Fundamentos de Python
- Script inicial
- Fundamentos del lenguaje
- Estructuras de control
- Estructuras de datos
- Control de avance

### Módulo 2: Temas avanzados
- Funciones
- Manejo de errores y excepciones
- Entrada y salida de datos
- Uso de módulos
- Paquetes y organización de código
- Control de avance

## Archivos incluidos
- `datos.txt`, `datos2.txt`: Archivos de datos para ejercicios prácticos
- Carpeta `modulo 1/`: Scripts y ejercicios del primer módulo
- Carpeta `modulo 2/`: Scripts y ejercicios del segundo módulo, incluyendo ejemplos de paquetes

## Requisitos
- Tener instalado Python (recomendado versión 3.7 o superior)
- Editor de texto o IDE (Visual Studio Code recomendado)

## Objetivos de aprendizaje
- Comprender los conceptos básicos de programación
- Aprender la sintaxis y las estructuras principales de Python
- Desarrollar scripts y programas sencillos
- Manejar errores y excepciones
- Trabajar con módulos y paquetes

## Cómo usar este repositorio
1. Clona o descarga el repositorio en tu computadora.
2. Abre los scripts de cada módulo y sigue las instrucciones y ejemplos.
3. Utiliza los archivos de datos para practicar la lectura y manipulación de información.
4. Consulta los scripts de control de avance para verificar tu progreso.
